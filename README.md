# GRANO DURO

Luxurious single-page website for **GRANO DURO** — an intimate Italian restaurant in Powiśle, Warsaw.

## Stack

- React 19
- Vite 8

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Sections

- Hero with brand identity
- Philosophy & handmade pasta story
- Weekly menu
- Signature dishes (carbonara, ravioli, risotto, bruschetta, cannolo)
- Chef Daniele Noè
- Wine pairing
- Guest reviews carousel
- Visit, hours & reservation
